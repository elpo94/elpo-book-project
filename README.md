# 사부작 (Sabujak)
> 일상의 작은 몰입을 돕는 사려 깊은 스케줄러와 타이머 

'사부작'은 거창한 계획에 압도되기보다, 지금 이 순간의 작은 집중이 삶의 습관으로 스며들 수 있도록 돕는 서비스입니다. 

## 주요 기능 (Key Features)
몰입 타이머: 현재 작업에 온전히 집중할 수 있는 미니멀한 타이머 
프로젝트 관리: 작은 단위의 작업을 생성하고 관리하는 유연한 시스템 
기록 캘린더: 사부작히 쌓아온 몰입의 흔적을 시각적으로 확인 
철저한 익명성: Firebase 익명 로그인을 통한 안전한 데이터 동기화 

## 🛠 기술 스택 (Tech Stack)
Framework: Flutter (Dart)
Database/Auth: Google Firebase (Firestore, Anonymous Auth) 
Design: Figma (Beige-toned Warm UI) 

## 🚀 시작하기 (Getting Started)
1. Flutter 환경 설정 확인 
2. `google-services.json` 설정 (Firebase) 
3. `flutter pub get` 및 실행
